Контрольные вопросы

  1. Назначение формы – это структурирование ввода данных от пользователя и отправка этих данных на сервер для обработки.
  2. Элемент формы, служащий для ввода пароля – это <input type="password">.
  3. Элемент формы, служащий для создания меню – это <select> с вложенными <option>.
